## ✨ Features

* **Dynamic Random Wallpapers**: Integrated with the author's exclusive API `acgapi.dpdns.org`, providing random anime background images.
* **Interactive Magic Circle**: A magic circle in the bottom-right that expands on hover and triggers a "Mana Injection" loading animation when clicked.
* **Responsive Design**: Compatible with both PC and mobile devices, with optimized dimensions for the magic circle.

## 🚀 Quick Start

Simply copy the contents of `404.html` from this project into your own project's 404 error page.

### Source of Magic (API Reference)

> **Causality Reset**: By clicking the magic circle, you invoke a mana surge that reshapes the current reality.

* **API Address**: `https://acgapi.dpdns.org`
* **Mechanism**: By appending a timestamp `?t=${Date.now()}` to the URL, it forces the browser to bypass the cache, allowing an instant background switch.
